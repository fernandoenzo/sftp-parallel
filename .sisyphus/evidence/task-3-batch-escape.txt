# Task 3: sftp_escape() and batch command generation - Evidence

## Implementation

### sftp_escape(path: str) -> str
- Escapes backslashes: \ -> \\
- Escapes double quotes: " -> \"
- Returns escaped string

### build_batch_commands(remote_dir: str, local_dir: str, files: list[str]) -> str
- Starts with cd "escaped_remote_dir"
- For each file: put -f "escaped_local_path"
- Ends with bye
- Returns complete batch command string

## QA Verification

### sftp_escape tests
```
'file with spaces.txt'
'file\\\\backslash.txt'
'file\\"quote.txt'
```

### build_batch_commands test
```
cd "/remote"
put -f "/local/file1.txt"
put -f "/local/file2.txt"
bye
```
