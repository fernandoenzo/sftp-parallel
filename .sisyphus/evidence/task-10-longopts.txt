───────────────────────────────────────────────────────────
sftp-parallel-upload.sh
Uploads all files from a local folder to a remote SFTP server
using `put -f` (fsync) with configurable parallelism.

Each parallel slot runs its own sftp session with a batchfile
of `put -f` commands, so every file is fsynced on the server.

Usage:
  sftp-parallel-upload.sh [-j N] [-r remote_dir] user@host local_dir
  sftp-parallel-upload.sh [--jobs=N] [--remote-dir=remote_dir] user@host local_dir

  -j N, --jobs=N       Parallel sftp sessions (default: 2)
  -r remote_dir, --remote-dir=remote_dir  Remote directory (default: .)
  -h, --help           Show this help
